Este projeto foi pensado para ser um convite de aniversário interativo para uma festa de aniversário de 1 ano. Como recursos adicionais e interativos 
foi adicionada a localização do Google Maps, e botões de confirmação de presença via whatsapp. 
Foi projetado de acordo com o tema da festa do cliente e , para ter todo o dinamismo que a temática pedia, foram implementadas diversas animações no CSS, 
simulando a órbita dos elementos.
Para conferir, acesse https://josemiguel.vercel.app/
